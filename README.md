# How to use

## Run Service
    $ sudo docker-compose up -d

>  To rebuild this image

    $ docker-compose up --build


## Stop Service

    $ sudo docker-compose down


## Account
    u: admin
    p: itadmin


## android list sdk

    $ android list sdk -a -e
